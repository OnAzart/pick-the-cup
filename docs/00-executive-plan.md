# Executive plan: quick money from Pick The Cup

**Window:** today (July 1) to the Final (July 19) — 18 days. Round of 16 starts July 7, quarterfinals July 11, semifinals July 15.

## Reframe the core assumption

"More traffic = more money" is only true if a payment rail is *already live* when the traffic hits. Right now none are — the sponsor slot is a mock checkout, no ads are installed, no tip jar exists. A traffic spike into an unmonetized site converts to $0 and a wasted spike, since those visitors won't return after the Final. **Get one payment rail working before pushing for traffic**, not after.

SEO is correctly written off — a zero-authority site cannot rank against ESPN/CBS Sports in 18 days regardless of effort (see 02). Redirect that time into Reddit/YouTube Shorts/community seeding instead, which reward relevance over domain age.

## Day 1–2 (today/tomorrow)

- [ ] Ship match dates/kickoff countdown on the bracket (near-zero effort, unlocks urgency messaging for everything else).
- [ ] Ship a lightweight "Challenge link" share flow — reuse the existing share-card image pipeline, no new group/DB model needed. This is the realistic 48-hour version of "private leagues."
- [ ] Start Stripe/Ko-fi identity verification now — it's the one multi-day dependency in the whole plan, so it can't wait.
- [ ] Start cold outreach for real $100 sponsor slots (target: 20+ prospects — sports bars, football gear shops, fantasy-sports tools, regional brands). Close via Stripe Payment Link, not the mock flow.
- [ ] Film 3–5 "I let [AI/creator] predict the whole World Cup" videos using the app live, plus reaction content to today's real Round of 32 results.

## Day 3–5

- [ ] Ship simple flat-points scoring (prerequisite for the leaderboard).
- [ ] DM 15–20 micro football-content creators (10K–100K followers) for duet-style "predict with me" posts — not scripted ads.
- [ ] Post daily reaction/upset content with the champion card visible on-screen.
- [ ] Seed genuinely useful posts in r/soccer, r/worldcup, and country subreddits (this doubles as SEO — Reddit threads can rank within weeks even though your own domain can't).
- [ ] Apply to Media.net as an ad-revenue floor (days-long review, no hard traffic minimum — AdSense/Ezoic are not viable this cycle, see 03).

## Week 2 (Round of 16 onward, July 7+)

- [ ] Ship the leaderboard — the single biggest retention/virality feature (return-visit trigger every time a round locks).
- [ ] Run TikTok Spark Ads behind whichever organic post is showing traction — cold organic virality without a seed audience essentially doesn't happen.
- [ ] Close sponsor deals; get Ko-fi live and tied explicitly to the existing charity-partner strip.
- [ ] Ship "highlight what's wrong" (lower priority — retention feature, less viral than the leaderboard).

## Hard no's this cycle

- **AdSense / Ezoic** — review timelines (days to months) risk outlasting the tournament; Ezoic now requires 250K monthly users regardless.
- **Sports-betting affiliate deals** — real per-state licensing exposure ($500–$10K+ fees in some states). Do not add without dedicated legal review.
- **Misleading clickbait titles/thumbnails** — both TikTok and YouTube actively detect and throttle title/content mismatches; this backfires specifically during the window when reach matters most.
